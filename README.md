# Digital Agency Website

A responsive multi-page digital agency website built with semantic HTML, modular CSS architecture, and vanilla JavaScript. view the site: https://squareup-site.vercel.app

This project was my transition from building single-page layouts to structuring and managing a larger front-end codebase with reusable components and scalable styling patterns.

## Features

- Responsive multi-page layout
- Reusable navigation system
- Mobile menu interaction
- Custom range slider component
- Implicit CSS grid layout with scalable row and column borders
- Modular JavaScript structure
- Semantic and accessible HTML structure

## What I Learned

### 1. CSS Architecture

Working on multiple pages forced me to think beyond writing isolated styles.

I learned how to:
- Organize styles into reusable modules
- Separate layout, utilities, and component styles
- Avoid style collisions and unnecessary specificity
- Structure CSS for scalability and maintainability

### 2. Scalable Grid Border System

One of the most interesting challenges was creating a grid layout where both row and column borders scale automatically regardless of how many items exist in the grid.

This pushed me to better understand:
- Explicit vs implicit grids
- Grid auto-placement
- Pseudo-elements and border strategies
- Responsive layout behavior

### 3. Custom Range Slider

I built a custom interactive range slider using HTML, CSS, and JavaScript.

Through this, I learned:
- Styling native range inputs
- Synchronizing UI feedback with JavaScript
- Handling dynamic positioning
- Improving user interaction and visual feedback

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
